🖐️ What is Hand Gesture Recognition?

It is a system that detects and interprets human hand gestures using a camera.

Example:

✋ Open hand → Stop

👍 Thumbs up → Like

✌️ Two fingers → Peace

👊 Fist → Action trigger

Used in:

Virtual mouse

Sign language detection

Gaming

Smart home control

# 🖐️ Hand Gesture Recognition System

A real-time Hand Gesture Recognition system developed using Computer Vision techniques to detect and interpret hand gestures through webcam input.

The system uses image processing and machine learning to recognize gestures and perform predefined actions.

---

## 🚀 Features

- 🎥 Real-time webcam detection
- 🖐️ Hand landmark detection
- 🤖 Gesture classification
- ⚡ Fast processing
- 📊 High accuracy recognition
- 🔄 Action triggering based on gestures

---

## 🛠️ Tech Stack

### AI / Computer Vision
- Python
- OpenCV
- MediaPipe
- NumPy

### Optional Backend Integration
- Java Spring Boot (for storing gesture logs)
- MySQL

---

## 🏗️ How It Works

1. Capture video from webcam
2. Detect hand landmarks using MediaPipe
3. Extract key points
4. Classify gesture
5. Trigger action
